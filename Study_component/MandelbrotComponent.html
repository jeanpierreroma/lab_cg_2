What is fractal?

A fractal is a complex patterns that are self-similar across different scales. They are created by repeating a simple process over and over in an ongoing feedback loop. What is Mandelbrot fractal?

The Mandelbrot set is the set of complex numbers c for which the function f(z)=z*z+c does not diverge to infinity when iterated from z=0, i.e., for which the sequence f(0), f(f(0)) , remains bounded in absolute value. Images of the Mandelbrot set exhibit an elaborate and infinitely complicated boundary that reveals progressively ever-finer recursive detail at increasing magnifications; mathematically, one would say that the boundary of the Mandelbrot set is a fractal curve.
